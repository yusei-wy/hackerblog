ハミング距離とは

等しい文字数を持つ二つの文字列の中で、対応する位置にある異なった文字の個数 出典：ハミング距離

例えば

1001001
1000101

このハミング距離は”2”

2239567
2459587

このハミング距離は”3”

helloworld
hollndorld

このハミング距離は”3”

求め方

ソースコードで表すと

str1 = "1001001"
str2 = "1000101"

cnt = 0
for s1, s2 in zip(list(str1), list(str2)):
    if s1 is not s2:
        cnt += 1
print(cnt)  # 2

簡単ですね。
